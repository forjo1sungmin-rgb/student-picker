# Student Picker 🎲

React 기반으로 제작된 **학급 자리 뽑기 & 랜덤 학생 선택 웹앱**입니다.  
좌석 배치, 랜덤 호출, 그룹 나누기 등을 간단하게 처리할 수 있도록 설계되었습니다.

## ✨ 주요 기능
- 랜덤 학생 선택
- 좌석 배치도 생성 및 관리
- 학생 목록 추가/편집/삭제
- 필터링 또는 조건부 선택 기능 (추가 예정)
- 모바일 화면에서도 사용 가능한 인터페이스

## 🛠 기술 스택
- **React**
- **JavaScript / JSX**
- **Vite 또는 CRA 기반 개발** (프로젝트 설정에 맞게 수정 가능)
- **CSS / Tailwind / Styled Components 중 선택 가능**  
  (현재 프로젝트에서 사용한 방식으로 수정하세요.)
